# Fig3_pop_structure_dem_inference

### fasttree
Scripts used to generate tree file for all samples.

### gone2
Scripts used to run GONE2 to estimate recent changes in Ne

### pairwise_fst_pops
Scripts used to quantify Fst for each pairwise combination of populations

### pca_pops
Files used to calculate PCA across all samples in Plot_fig3.R

### phlash
Scripts used to run phlash to estimate historic changes in Ne

### Plot_fig3.R
Script used to produce Figure 3 of the main text.
