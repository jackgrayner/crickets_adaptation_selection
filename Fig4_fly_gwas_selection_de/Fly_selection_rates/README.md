## Fly selection rates

### flytrapping.csv

Results from fly trapping experiments.

- month: mont sampled
- site: site code (island.2-letter-identifier)
- island: island
- date: date sampled
- rain: whether/how much rain was recorded
- trial: trial number (per date)
- time: time trial started
- time_postsunset: time trial started relative to sunset
- trap: trap ID (by position in grid)
- playback: whether song/control playback
- flies_caught: number of flies captured in trap
